# 🎰 Stake Tracker - Trade Tracker

A professional trading/gambling session tracker with stop loss and target limits.

## ✨ Features

- **Session Management**: Set daily stop loss and target percentages
- **Bet Tracking**: Place chips with various denominations (₹10, ₹50, ₹100, ₹500, ₹1000)
- **Real-time Stats**: Track wins, losses, P&L, win rate, and expected value
- **Visual Alerts**: Animated notifications when limits are hit
- **Data Persistence**: Automatic CSV storage of all sessions
- **Undo Functionality**: Revert last win/loss action
- **Session Notes**: Add notes to remember key moments
